# c-fundamentals

# Overview

This repository contains basic to intermediate C programming programs created for learning programming fundamentals, logic building, and problem-solving.

The main goal of this repository is to strengthen core programming concepts used in software development, embedded systems, and technical interviews.

It includes beginner-friendly C programs covering conditions, loops, functions, arrays, patterns, number logic, and basic algorithms.

# Features

Beginner-friendly C programs
Logic-building exercises
